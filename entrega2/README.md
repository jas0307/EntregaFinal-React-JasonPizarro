##
Abra la carpeta frontend en vscode u otro editor de código y ejecútelo en el directorio raíz:
  
   ```bash
   npm install
   ```
Instale todos los paquetes. Una vez completada la instalación, ejecute en el directorio raíz

   ```bash
   npm run dev
   ```
